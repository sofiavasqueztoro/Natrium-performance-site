# Natrium Wallet — Performance Analysis Site

This project is a Vite + React site presenting a professional performance profiling dashboard for the Natrium Wallet mobile app.

Quick start

1. Install:

```bash
npm install
```

2. Run locally:

```bash
npm run dev
```

3. Build:

```bash
npm run build
```

Deployment

- GitHub Pages: build and push `dist` to `gh-pages` branch or use `gh-pages` package.
- Vercel / Netlify: connect repository and deploy using the default build command `npm run build`.

Replace placeholder images under `src/assets/` with real screenshots. See `src/components/ScreenshotCard.jsx` for where to plug in images.
